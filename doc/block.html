izi-pygame Documentation (Block)

The Block object is an invisible box that serves as the mother class of the Drawblock and the Picblock.

Class attributes

  1. Block.nb_block -> The number of instances created
  2. Block.list_block -> A list that contain all the instances created

Instance attributes

  1. self.xbegin -> x origin (left side) of the instance
  2. self.ybegin -> y origin (up side) of the instance
  3. self.xend -> the opposite of self.xbegin (right side) of the instance
  4. self.yend -> the opposite of self.ybegin (bottom side) of the instance
  5. self.width -> the width of the instance
  6. self.height -> the height of the instance
  7. self.speed -> the speed of the instance (not used if the instance doesn't move())
  8. self.id -> the identification code of the instance (equal to the Block.nb_block at the cration of the instance)

__init__ method

def __init__(self, x=0, y=0, width=100, height=100, speed=5)
the current instance is added to the Block.list_block and Block.nb_block is incremented

Methods

  1. def set_position(self, x, y) -> set the position of the instance
  2. def set_mouse_position(self, pos="center") -> set the position of the instance with the mouse position. (pos="center" or "origin"[mx,my])
  3. def move(self, direction, spd=None) -> move the instance in a direction with a speed.
    if speed is not mentionned, speed=self.speed. direction can take 4 values ("right" "left" "up" "down")
  4. def gravity(self, g=10) -> the instance is influenced by the gravity
  5. def delete_from_class_list(self) -> the instance is deleted from the Block.list_block
  6. def cursor_collide(self, cursor) -> detect a collision with a block 1x1
    return boolean
  7. def press_cursor_collide(self, cursor) -> detect a collision with a block 1x1 and a left click at the same time
    return boolean
  8. def mult_size(self, fact=1) -> the size of the instance is mult by fact
  9. def div_size(self, fact=1) -> the size of the instance is divided by fact
  10. def pow_size(self, power=2) -> the size of the instance is powered by fact

Specials methods

  1. def __imul__(self, fact) -> same as def mult_size(self, fact=1) but with *=
  2. def __itruediv__(self, fact) -> same as def div_size(self, fact=1) but with /=
  3. def __ifloordiv__(self, fact) -> same as def div_size(self, fact=1) but with //=
  4. def __ipow__(self, power) -> same as def pow_size(self, power=2) but with **=
  5. def __lshift__(self, other) -> the size of the instance is added with the size of an other instance with <<
  6. def __rshift__(self, other) -> the size of the other instance is added with the size of the current instance with >>
  7. def __str__(self) -> return a string that contains witdh/height and x/y
