izi-pygame Documentation (Spriteblock)

The Spriteblock object is a Picblock that can animated itself with a list of picture.

Instance attributes

  1. self.sprite_list -> a list of pictures
  2. self.index_picture -> the index of the current picture in the animation
  3. self.number_pictures -> the lenght of self.sprite_list
  4. self.frequence -> speed of animation (not the same with each fps)

__init__ method

def __init__(self, sprites, window, first_sprite=0, x=0, y=0, width=100, height=100, speed=0, frequence=10)

Methods

  1. def anime(self, jump=1) -> print and anime the instance (jump a number of pictures to each call of this method)
  2. def set_sprite_list(self, sprites) -> set the sprite_list
