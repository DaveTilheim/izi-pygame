izi-pygame Documentation (Picblock)

The Picblock object is a Block object that can be print on a pygame window with a picture

Instance attributes

  1. self.namepic -> the name of the picture to print
  2. self.window -> the pygame.Surface where the instance will be printed
  3. self.pic -> pygame.image

__init__ method

def __init__(self, x=0, y=0, width=100, height=100, speed=0, namepic=None, window=None)

Methods

  1. def print(self) -> print the instance to the self.window
  2. def set_pic(self, pic, width=None, height=None) -> set the picture of the instance
