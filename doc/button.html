izi-pygame Documentation (Button)

The Button object is a Block object. You can attribute it a callback function which is activate if the user click on the button

Class attributes

  1. Button.list_button -> The list of all Button instances

Instance attributes

  1. self.cursor -> a Cursor object
  2. self.sensitive -> true if the button is clickable
  3. self.ajust -> true if the text is centralized
  4. self.ajust_value -> padding value

__init__ method

def __init__(self, callback, cursor, x=0, y=0, width=100, height=50, text=None, text_color=(0,0,0), window=None, font=None, ajust=False, ajust_value=0)

Methods

  1. def get_text(self) -> get the text of the button
  2. def set_text(self) -> set the text of the button
  3. def set_sensitive(self, boolean) -> set the sensitivity of the button
  4. def ajust_button(self) -> ajust the position and the dimension of the button
  5. def launch(self) -> print the button and wait detection
