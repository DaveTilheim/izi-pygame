izi-pygame Documentation (Entry)

The Entry object is a Button object who you can entry a piece of text

Instance attributes

  1. self.max_char -> the maximum of char who you can write on an Entry

__init__ method

def __init__(self, x=0, y=0, width=100, text=str("your text here"), font=None, window=None, cursor=None, max_char=100, text_color=(0,0,0))

Methods

  1. def focus_callback(self) -> get the focus of the entry
  2. def launch(self) -> print the entry
  3. def grab_key(self, key_name) -> catch a keyname thanks to the user and update the entry
