izi-pygame Documentation (Drawblock)

The Drawblock object is a Block object that can be draw on a pygame window

Instance attributes

  1. self.color -> the color of the instance
  2. self.window -> the pygame.Surface where the instance will be drawed
  3. self.fill -> if 0 indicates that the instance will be filled with color

__init__ method

def __init__(self, x=0, y=0, width=100, height=100, speed=5, color=(255,255,255), fill=0, window=None)

Method

def draw(self, form="rect") -> draw the instance to the self.window. form="rect" or "circle"
