izi-pygame Documentation (Window)

The Window object is an object that manage the pygame.Surface to print pygame objects

Instance attributes

  1. self.wtitle -> the title of the window
  2. self.wwidth -> the width of the window
  3. self.wheight -> the height of the window
  4. self.win -> the pygame window

__init__ method

def __init__(self, wwidth=500, wheight=500, wtitle="unknown")

Methods

  1. def fill(self, color) -> the background of the window is filled with color
  2. def get_canva(self) / surface(self) -> return self.win
  3. def set_size(self, wwidth=500, wheight=500) -> set the size of the window
  4. def set_title(self, string) -> set the title of the window
