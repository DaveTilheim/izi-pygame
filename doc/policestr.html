izi-pygame Documentation (Policestr)

The Policestr object is a string that can be write on the screen with a pygame.Surface object

Instance attributes

  1. self.police -> the pygame.font
  2. self.window -> the pygame.Surface where the instance will be drawed
  3. self.policename -> the name of the police
  4. self.policesize -> the size of the police
  5. self.txt -> the text to print
  6. self.string -> the final text ready to write in the screen with the pygame.font
  7. self.x -> the x position of the instance
  8. self.y -> the y position of the instance

__init__ method

def __init__(self, name=None, size=20, string="text", color=(0,0,0), window=None, x=0, y=0, italic=0, bold=0, underline=0)

Methods

  1. def refresh(self) -> update the self.string with new modifications
  2. def write(self) -> write the instance to the screen self.window
  3. def set_text(self, string) -> set the text to write to the screen
  4. def get_text(self) -> return self.txt
  5. def set_font(self, name=None, size=None) -> set the font of the instance
  6. def set_style(self, it=False, bd=False, ul=False) -> set the style of the text printed
  7. def strcat(self, string) -> concatenation of the self.txt with string

Specials methods

  1. def __rshift__(self, string) -> same as def strcat(self, string) but with >>
  2. def __lshift__(self, string) -> set the text to print with <<
  3. def __iadd__(self, fact) -> up the police size with +=
  4. def __isub__(self, fact) -> down the police size with -=
  5. def __imul__(self, fact) -> mult the police size with *=
  6. def __itruediv__(self, fact) -> div the police size with /=
  7. def __ifloordiv__(self, fact) -> div the police size with //=
  8. def __str__(self) -> same as get_text(self)
